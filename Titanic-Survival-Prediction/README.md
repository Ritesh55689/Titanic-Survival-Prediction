# Titanic Survival Prediction

## Objective
Build a machine learning model to predict whether a passenger survived the Titanic disaster.

## Dataset
The Titanic dataset contains passenger information such as:
- Passenger Class
- Age
- Gender
- Fare
- Cabin
- Embarked
- Survival Status

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

## Machine Learning Algorithm
- Random Forest Classifier

## Steps Performed
1. Loaded the dataset
2. Cleaned missing values
3. Encoded categorical features
4. Split the data into training and testing sets
5. Trained the Random Forest model
6. Evaluated the model using accuracy, confusion matrix, and classification report

## Result
The model successfully predicts passenger survival with an accuracy of approximately 80–85%.
