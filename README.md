# CODSOFT Machine Learning Internship

Welcome to my **CodSoft Machine Learning Internship** repository. This repository contains the projects completed as part of the internship, demonstrating the application of machine learning techniques for solving real-world problems using Python and Scikit-learn.

## Internship Tasks

### Task 1 – Titanic Survival Prediction

**Objective:**
Build a machine learning model to predict whether a passenger survived the Titanic disaster based on passenger information such as age, gender, passenger class, fare, and family details.

**Algorithm Used:**

* Random Forest Classifier

**Key Concepts:**

* Data Cleaning
* Missing Value Handling
* Feature Encoding
* Classification
* Model Evaluation

---

###  Task 2 – Movie Rating Prediction with Python

**Objective:**
Develop a regression model to predict movie ratings using features such as genre, director, actors, duration, year, and number of votes.

**Algorithm Used:**

* Random Forest Regressor

**Key Concepts:**

* Data Preprocessing
* Feature Engineering
* Regression
* Model Evaluation
* Data Visualization

---

### Task 3 – Credit Card Fraud Detection

**Objective:**
Build a machine learning model to identify fraudulent credit card transactions by preprocessing data, handling class imbalance, and classifying transactions as genuine or fraudulent.

**Algorithm Used:**

* Logistic Regression

**Key Concepts:**

* Data Normalization
* Handling Imbalanced Data
* Oversampling
* Binary Classification
* Precision, Recall, and F1-Score Evaluation
