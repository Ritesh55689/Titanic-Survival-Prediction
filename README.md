# CODSOFT Machine Learning Internship

This repository contains all the tasks completed as part of the CodSoft Machine Learning Internship.

## Tasks

### Task 1 - Titanic Survival Prediction
- Predict whether a passenger survived the Titanic disaster using machine learning.

### Task 2 - Movie Rating Prediction
- Predict movie ratings using features such as genre, director, actors, duration, year, and votes.

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- Jupyter Notebook
