#Movie Rating Prediction
This project is part of the CodSoft Machine Learning Internship.

##Objective
Build a machine learning model that predicts movie ratings using features such as genre, director, actors, duration, year, and votes.

##Technologies Used
-Python
-Pandas
-NumPy
-Matplotlib
-Scikit-learn

##Machine Learning Algorithm
-Random Forest Regressor

##Workflow
-Data Cleaning
-Data Preprocessing
-Feature Encoding
-Train-Test Split
-Model Training
-Prediction
-Model Evaluation

##Evaluation Metrics

-Mean Absolute Error (MAE)
-Mean Squared Error (MSE)
-Root Mean Squared Error (RMSE)
-R² Score
